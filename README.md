# CircuitForge 🚀

CircuitForge is a full-stack electronics project management platform built for engineering students. It streamlines the entire lifecycle of custom electronics project development—from project submission and admin review to realtime progress tracking, report delivery, and hardware handoff.

### ✨ Features

* 🔐 Secure Student Authentication (Supabase Auth)
* 📤 Project Submission Portal
* 📋 Admin Project Management Dashboard
* 🔔 Realtime Notifications
* 📊 Live Project Status Tracking
* 📁 File Upload & Deliverable Management
* 📄 PDF Report & Documentation Delivery
* 🎥 Demo Video Delivery
* 📦 Hardware Delivery & Pickup Tracking
* ⚡ Supabase Realtime Integration
* 🛡️ Role-Based Access Control
* 📈 Analytics & Project Monitoring

### 🏗️ Tech Stack

**Frontend**

* React
* Vite
* Tailwind CSS
* React Router
* Framer Motion
* Lucide React

**Backend**

* Node.js
* Express.js

**Database & Backend Services**

* Supabase PostgreSQL
* Supabase Auth
* Supabase Realtime
* Supabase Storage

### 🎯 Vision

CircuitForge aims to bridge the gap between engineering students and practical project development by providing a centralized platform where students can submit project requirements, track progress in realtime, communicate with the development team, and receive complete project deliverables before their academic deadlines.

### 🌟 Core Workflow

Student → Submit Project → Admin Review → Project Development → Realtime Updates → Report & Video Delivery → Hardware Handoff

Built with ❤️ to simplify electronics project development and management.
